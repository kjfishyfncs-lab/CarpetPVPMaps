About the pack

Built for carpet, tuned for combat.

Every map is designed around one idea: clean, readable PvP where the terrain never gets in the way of the fight.

ItzVortex CarpetPVP Maps is a map pack made for Carpet-mod servers running fast, no-nonsense combat. No clutter, no lag machines, no builds that break hit-detection — just tight arenas built around movement, timing, and knockback.

Every arena ships in the same light-blue carpet-and-ice palette so the pack feels like one connected world instead of a pile of unrelated schematics. Floors are flat, walls are clean, and cover is placed on purpose — not left over from decoration.

New maps get play-tested in-house before release, and the Discord is where testing calls, map requests, and schematic drops happen first.

What's inside the pack

  • Box fight arenas sized for 1v1 and 2v2
  • Bridge maps built around rotation and spacing
  • Sky arenas with edge-play and knockback zones
  • Free-for-all pits for warmups and events
  • Downloadable schematics for server owners
